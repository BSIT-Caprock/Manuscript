# Manuscript

Contains Capstone manuscript paper based on a [custom template](https://github.com/BSIT-Caprock/Manuscript-Template) with quick styles and advanced formatting options. 

## Our workflow

1. Mark the task that is currently being worked on as `DOING`.
2. Go to Google Docs and use it as a scratchpad for collaborative writing.
3. Proofread and review the new writing. Use the built-in tools in Docs.
4. Open the manuscript document in Microsoft Word.
5. Copy and paste content from the scratchpad to the Word document.
6. Review the pasted content.
7. Save the document and commit the changes to git (if applicable).
8. Mark the finished task as `DONE`.
